AI MediaKit 提供**字幕擦除（标准版）和字幕擦除（精细化版）** 两个工具，实现对视频画面中内嵌字幕（硬字幕）的自动化检测与擦除。


* **标准版**：提供高效、经济的字幕擦除方案，适用于对成本和速度有较高要求的通用场景。

* **精细化版**：依托火山引擎高帧率 OCR 与 AIGC 图像修复算法，实现高质量的无痕擦除，最大程度地还原视频原始画面，适用于短剧出海、专业二创等对画质要求极高的场景。


本指南将帮助您了解两个版本的功能区别，并快速上手，通过 API 实现视频字幕的自动化处理。

## 快速体验：一键擦除短剧字幕

本节将通过一个具体的示例，演示如何使用**字幕擦除（精细化版）** 工具，快速实现针对**中英文字幕短剧场景**的字幕自动检测与擦除。

<span id="gFaEwYzzZ9"></span>
### 准备工作


1. **获取访问凭证**：在开始之前，访问 AI MediaKit 控制台 [API Key 管理页面](https://console.volcengine.com/imp/ai-mediakit/settings)，创建并获取 **AI MediaKit API Key**（用于媒体处理功能鉴权）。

2. **准备视频文件**：准备一个待处理的视频文件，并将其上传至公网可访问的服务器或[视频点播服务](https://www.volcengine.com/docs/4/2834?lang=zh)中，以获取可供 API 调用的 URL。为方便您快速测试，本文后续步骤的示例代码中提供了一个可直接使用的演示视频 URL。


<span id="FJMsyyf92C"></span>
### 操作流程

精细化字幕擦除是一个异步任务，整体流程如下：


1. 成功调用 `POST /api/v1/tools/erase-video-subtitle-pro` 接口提交任务后，您将获得一个唯一的 `task_id`。

2. 您可以轮询 `GET /api/v1/tasks/{task_id}` 接口，直到任务状态`status` 变为 `completed`；或者使用[事件回调](https://www.volcengine.com/docs/6448/2288701)自动接收任务状态变化。

3. 任务完成后，解析响应中的 `result` 字段，即可在 `video_url` 字段中获取处理后视频的下载地址。


<span id="M2mLmLyGtz"></span>
#### 第 1 步：提交任务

调用[提交字幕擦除（精细化版）任务 API](https://www.volcengine.com/docs/6448/2372084)。您可以直接复制以下 cURL 命令，向 API 提交一个字幕擦除任务。

```Bash
curl -X POST 'https://mediakit.cn-beijing.volces.com/api/v1/tools/erase-video-subtitle-pro' \
-H 'Authorization: Bearer {Your_API_Key}' \
-H 'Content-Type: application/json' \
-d '{
  "video_url": "https://vod-ai-test.tos-cn-beijing.volces.com/demo/demo_video_drama_hard_subs.mp4",
  "mode": "Subtitle"
}'
```


<div data-tips="true" data-tips-type="default" data-tips-is-title="true">说明</div>


<div data-tips="true" data-tips-type="default">需替换参数：</div>



* <div data-tips="true" data-tips-type="default">将 <code>{Your_API_Key}</code> 替换为您的有效 API Key。</div>


* <div data-tips="true" data-tips-type="default"><code>-d</code> 部分是 JSON 请求体，您可以直接修改其中的关键参数来自定义您的视频：</div>


   * <div data-tips="true" data-tips-type="default"><code>video_url</code>：示例中已提供可直接用于测试的演示视频 URL。您也可以将其替换为您自己的视频地址。</div>


   * <div data-tips="true" data-tips-type="default"><code>mode</code>：指定擦除模式为 <code>Subtitle</code>，擦除 OCR 检测为字幕的文本。此模式下，系统会自动识别并擦除满足以下条件的字幕：</div>


      * <div data-tips="true" data-tips-type="default"><strong>位置</strong>：位于视频画面下方 50% 以内，且在横向偏中央的位置。</div>


      * <div data-tips="true" data-tips-type="default"><strong>大小</strong>：文字竖向高度在视频高度的 1%～10% 之间。</div>


      * <div data-tips="true" data-tips-type="default"><strong>颜色</strong>：白色。</div>



请求成功后，您会收到如下包含 `task_id` 的响应，请保存此 ID 用于后续查询。

```JSON
{
  "success": true,
  "task_id": "amk-tool-erase-video-subtitle-pro-123456789",
  "request_id": "20240520203000ABCDEFG12345"
}
```


<span id="rxQEKvh3Mb"></span>
#### 第 2 步：获取结果

由于视频处理需要时间，您需要通过轮询[查询任务信息 API](https://www.volcengine.com/docs/6448/2278532) 来获取最终结果，直到任务状态变为 `completed`。

```Bash
curl -X GET 'https://mediakit.cn-beijing.volces.com/api/v1/tasks/{task_id}' \
-H 'Authorization: Bearer {Your_API_Key}'
```


<div data-tips="true" data-tips-type="default" data-tips-is-title="true">说明</div>


<div data-tips="true" data-tips-type="default">需替换参数：将 URL 末尾的 <code>{task_id}</code> 替换为第 1 步返回的任务 ID。</div>


当任务成功完成后，您将收到如下响应。`result` 字段中的 `video_url` 即是处理后视频的下载地址（默认输出为 **MP4** 格式，有效期为 **24** 小时）。

```JSON
    "task_type": "erase-video-subtitle-pro",
    "status": "completed",
    "result": {
        "video_url": "https://example.volcvideo.com/erased-video.mp4?auth_key=...",
        "duration": 180.5
    },
    "expires_at": 1775047666,
    "created_at": 1774874853,
    "finished_at": 1774874866,
    "request_id": "1742012dcc1938455d84dbe56beb7acc"
}
```

## 进阶使用（仅精细化版）

<span id="4cd87db9"></span>
### 指定区域擦除（框选擦除）

除了依赖 OCR 自动检测，您还可以通过 `erase_ratio_location` 参数手动指定一个或多个矩形区域。系统将强制擦除这些区域内的所有文本内容，这对于擦除短剧中常见的、位置固定的人名或地名等非常有用。每个区域的位置通过其左上角和右下角的坐标来定义，所有坐标值均为相对于视频总宽高的比例（取值范围 0.0 ～ 1.0）。假设您想要将下图橙色区域作为擦除区域，则参数设置如下：


* `top_left_x`: 矩形左上角的 X 坐标（图中 b1）。

* `top_left_y`: 矩形左上角的 Y 坐标（图中 a1）。

* `bottom_right_x`: 矩形右下角的 X 坐标（图中 b2）。

* `bottom_right_y`: 矩形右下角的 Y 坐标（图中 a2）。


<span>![图片](https://p9-arcosite.byteimg.com/tos-cn-i-goo7wpa0wc/166213c4d45145fcbae7c7b03aeb90d3~tplv-goo7wpa0wc-image.image) </span>

&nbsp;

例如，短剧视频中常在左上角或右上角显示人名、地名。以下配置通过定义两个独立的擦除框，同时覆盖这两个区域，确保区域内的所有文本都会被擦除。

<div data-tips="true" data-tips-type="default" data-tips-is-title="true">说明</div>


<div data-tips="true" data-tips-type="default"><code>mode</code> 设置为 <code>"Text"</code>，以确保系统会检测并擦除所有类型的文本，而不仅仅是字幕。</div>


```JSON
{
  "video_url": "https://example.com/short_drama_clip.mp4",
  "mode": "Text",
  "erase_ratio_location": [
    {
      "top_left_x": 0.05,
      "top_left_y": 0.05,
      "bottom_right_x": 0.25,
      "bottom_right_y": 0.15
    },
    {
      "top_left_x": 0.75,
      "top_left_y": 0.05,
      "bottom_right_x": 0.95,
      "bottom_right_y": 0.15
    }
  ]
}
```

## 参考信息

<span id="e9397274"></span>
### 核心优势


* **高帧率 OCR 字幕检测**：依托火山引擎高帧率 OCR 字幕检测算法，精准识别并定位视频中的字幕内容。该技术针对短剧等场景专门优化，不仅能准确区分需要擦除的字幕与需保留的剧情文本（如剧情提示、剧名、文字贴纸等），还采用高帧率抽帧技术确保无遗漏检测，为后续的精准擦除奠定坚实基础。

* **AIGC 图像修复**：在图像修复环节，采用自主研发的 AIGC 算法，结合时序分析与多帧联合处理，智能重建被擦除区域的画面。算法通过光流计算和特征匹配保持帧间稳定性，并运用深度学习生成逼真纹理，使修复后的画面与原始视频无缝融合，几乎看不出修改痕迹。


<span id="2f187a65"></span>
### 应用场景


* **二次创作**：在进行视频混剪或二次创作时，去除原视频字幕，以便根据新的创意添加自定义字幕或音频，使内容更符合新主题和风格。

* **内容翻译与出海**：将短剧等内容在海外发行时，需要先擦除原视频的中文字幕，再添加目标语言的字幕。反之，在引入海外内容时，也需要擦除外文字幕并替换为中文字幕。

* **AIGC 内容修正**：当 AIGC 生成的视频中出现非预期的文字或字幕时（负向约束失效），可利用本功能进行高质量的无痕擦除。

* **广告与营销**：根据不同平台的特性与受众需求调整广告视频。例如，擦除原有字幕，重新设计并制作与广告风格及宣传重点更匹配的新字幕，以增强视觉冲击力与信息传递效能。

<span id="0c3036a5"></span>
### 处理耗时

任务处理时长与视频的原始时长相关。RTF (Real\-Time Factor) 指处理耗时与原片时长的比值，可用于预估任务耗时。


* **对于时长大于 1 分钟的视频**：平均 RTF 为 6 ～ 10。例如，一个 10 分钟的视频，处理耗时约在 60 ～ 100 分钟之间。

* **对于时长小于 1 分钟的视频**：平均 RTF 约为 25。例如，一个 30 秒的视频，处理耗时约 750 秒，即 12.5 分钟。
