## 快速体验：生成一个 AI 解说视频

本节将演示如何通过“剧本还原”和“解说生成”两步，从多个原始视频快速生成一个使用 AI 自动创作解说词、并带有 AI 配音和字幕的解说视频。

<span id="CeyN4B0WED"></span>
### 准备工作


1. **获取访问凭证**：在开始之前，访问 AI MediaKit 控制台 [API Key 管理页面](https://console.volcengine.com/imp/ai-mediakit/settings)，创建并获取 **AI MediaKit API Key**。

2. **准备视频文件**：准备一个或多个短剧视频，并将其上传至公网可访问的服务器或[视频点播服务](https://www.volcengine.com/docs/4/2834?lang=zh)中，以获取可供 API 调用的 URL。为方便您快速测试，本文后续步骤的示例代码中提供了两个可直接使用的演示视频 URL。


<span id="AkglA3gjgF"></span>
### 操作流程

解说视频的生成是一个包含两个异步任务的流程，核心步骤如下：


1. **提交剧本还原任务并获取剧本还原任务 ID**：

   1. 调用 `POST /api/v1/tools/drama-script`，获得一个用于查询的 `task_id`。

   2. 轮询 `GET /api/v1/tasks/{task_id}` 直至任务完成，然后从响应的 `result` 字段中解析出 `drama_script_task_id`。

2. **提交解说视频生成任务并获取最终结果**：

   1. 调用 `POST /api/v1/tools/drama-recap` 并传入上一步获得的 `drama_script_task_id`，获得第二个用于查询的 `task_id`。

   2. 轮询 `GET /api/v1/tasks/{task_id}` 查询最终任务状态。任务完成后，从 `result` 字段中获取解说视频下载地址。


<span id="T6ULrSpcTR"></span>
#### 第 1 步：提交剧本还原任务

首先，调用[提交剧本还原任务 API](https://www.volcengine.com/docs/6448/2484442) 对您的原始视频进行分析。可直接复制以下 cURL 命令提交剧本还原任务。

```Bash
curl -X POST 'https://mediakit.cn-beijing.volces.com/api/v1/tools/drama-script' \
-H 'Authorization: Bearer {Your_API_Key}' \
-H 'Content-Type: application/json' \
-d '{
  "video_urls": [
    "https://vod-ai-test.tos-cn-beijing.volces.com/demo/demo_video_drama_rekindled_love_ep01.mp4",
    "https://vod-ai-test.tos-cn-beijing.volces.com/demo/demo_video_drama_rekindled_love_ep02.mp4"
  ]
}'
```


<div data-tips="true" data-tips-type="default" data-tips-is-title="true">说明</div>



* <div data-tips="true" data-tips-type="default">将 {Your_API_Key} 替换为您的真实 API Key。</div>


* <div data-tips="true" data-tips-type="default"><code>video_urls</code>：示例中已提供可直接用于测试的演示视频 URL。您也可以将其替换为您自己的视频地址。</div>



请求成功后，您会收到一个用于查询进度的 `task_id`（例如 `amk-tool-drama-script-123456789`）。

接下来，您需要使用此 `task_id` 轮询[查询任务信息 API](https://www.volcengine.com/docs/6448/2278532)。当任务状态变为 `completed` 后，从响应的 `result` 字段中，找到并保存 `drama_script_task_id` 的值。这个值（例如 `v03cd6g10067d8aotcchmjljad3qra7g`）是下一步所需的剧本还原任务 ID。

成功完成的剧本还原任务响应示例如下：

```JSON
{
    "success": true,
    "task_id": "amk-tool-drama-script-123456789",
    "status": "completed",
    "result": {
        "result_url": "https://...",
        "duration": 245.5,
        "drama_script_task_id": "v03cd6g10067d8aotcchmjljad3qra7g" // <--- 请保存此 ID
    },
    ...
}
```


<span id="hDzgFLfxsh"></span>
#### 第 2 步：提交解说视频生成任务

获取到上一步中的 `drama_script_task_id` 后，调用[提交解说视频生成任务 API](https://www.volcengine.com/docs/6448/2488167)，并传入该 ID。以下示例将指示 AI 自动生成“悬疑风格”的解说词，并合成配音和字幕。

```Bash
curl -X POST 'https://mediakit.cn-beijing.volces.com/api/v1/tools/drama-recap' \
-H 'Authorization: Bearer {Your_API_Key}' \
-H 'Content-Type: application/json' \
-d '{
  "drama_script_task_id": "v03cd6g10067d8aotcchmjljad3qra7g",
  "erase_subtitle": true,
  "drama_recap_config": {
    "auto_generate_recap": true,
    "style": "悬疑风格，层层递进，结尾留下巨大悬念",
    "text_speed": 1.2
  },
  "speaker_config": {
    "voice_type": "Yunfeng"
  }
}'
```


<div data-tips="true" data-tips-type="default" data-tips-is-title="true">说明</div>



* <div data-tips="true" data-tips-type="default">将 <code>{Your_API_Key}</code> 替换为您的有效 API Key。</div>


* <div data-tips="true" data-tips-type="default">将 <code>drama_script_task_id</code> 替换为您在第 1 步轮询结果中获得的、以 <code>v</code> 开头的真实剧本还原任务 ID。</div>



请求成功后，您会收到一个新的 `task_id`（例如 `amk-tool-drama-recap-abcdefghi`），请保存此 ID 用于查询最终的视频结果。

再次轮询[查询任务信息 API](https://www.volcengine.com/docs/6448/2278532)，直到任务状态 `status` 变为 `completed`。

```Plain Text
curl -X GET 'https://mediakit.cn-beijing.volces.com/api/v1/tasks/amk-tool-drama-recap-abcdefghi' \
-H 'Authorization: Bearer {Your_API_Key}'
```


当任务成功完成后，您将收到如下响应。您可以在 `result.video_url` 中找到最终生成的解说视频下载地址。

```JSON
{
    "success": true,
    "request_id": "2026052711093289E685E06E529500826C",
    "task_id": "amk-tool-drama-recap-121654832642",
    "task_type": "drama-recap",
    "status": "completed",
    "result": {
        "video_url": "https://example.volcvideo.com/output.mp4?auth_key=...",
        "video_urls": [
            "https://example.volcvideo.com/output.mp4?auth_key=..."
        ],
        "duration": 29.314,
        "total_count": 1,
        "success_count": 1,
        "failed_count": 0,
        "error": {}
    },
    "created_at": 1779851373,
    "finished_at": 1779852072,
    "expires_at": 1779938471
}
```

## 进阶使用

在快速入门的基础上，您可以通过在提交解说视频生成任务时配置不同的参数，实现更丰富的创作效果。

<span id="0f982448"></span>
### 自定义解说词

您可以通过以下两种方式控制解说词：


* **AI 自动生成**：设置 `"auto_generate_recap": true`，并可通过 `style`, `text_length` 等参数影响 AI 的创作风格和长度。

* **使用自定义文案**：设置 `"auto_generate_recap": false`（或不传此参数），并通过 `recap_text` 字段提供您自己的解说文案。


<span id="7b3907dd"></span>
#### 示例：让 AI 自动生成 500 字左右的搞笑风格解说词

```JSON
{
  "drama_script_task_id": "v03cd6g10067d8aotcchmjljad3qra7g",
  "drama_recap_config": {
    "auto_generate_recap": true,
    "style": "风格轻松搞笑，多用网络热梗",
    "text_length": 500
  }
}
```


<span id="09cad164"></span>
#### 示例：使用您自己的解说词

```JSON
{
  "drama_script_task_id": "v03cd6g10067d8aotcchmjljad3qra7g",
  "recap_text": "故事的开头，男主角在街角遇到了改变他一生的人...",
  "drama_recap_config": {
    "auto_generate_recap": false
  }
}
```


<span id="bba501b2"></span>
### 自定义 AI **配音音色**

通过 `speaker_config.voice_type` 参数，您可以为 AI 配音选择不同的音色。下表详细介绍了各款预置音色的性别和风格特点。


|音色代码 (`voice_type`) |性别 |风格特点 |
|---|---|---|
|Yunfeng (云枫) |男声 |自信激昂，富有感染力，充满能量感与深度。 |
|Yunjian (云健) |男声 |音色深沉，风格休闲，兼具专业感与亲和力。 |
|Yunze (云泽) |男声 |音色深沉，沉稳自信，风格正式、严肃。 |
|Yunye (云野) |男声 |声音成熟放松，情感丰富多变。 |
|Yunxi (云希) |男声 |声音活泼阳光，情感表现力强，富有活力。 |
|Yunyi (云逸) |男声 |音色温柔，风格休闲友好，语调平静亲切。 |
|Yunjie (云杰) |男声 |自信干练，音色温暖，兼具专业感与亲和力，值得信赖。 |
|Xiaoxiao (晓晓) |女声 |声音活泼温暖，情感丰富，风格百搭多变。 |

<span id="490fbb64"></span>
#### 示例：使用 Yunjian 音色

```JSON
{
  "drama_script_task_id": "v03cd6g10067d8aotcchmjljad3qra7g",
  "recap_text": "...",
  "speaker_config": {
    "voice_type": "Yunjian"
  }
}
```


<span id="8a3c4dfc"></span>
### 调整 AI 配音语速与节奏

为了让解说更具吸引力，您可以通过以下参数精细控制配音的节奏感：


* `drama_recap_config.text_speed`：控制配音语速。取值范围为 `[0.5, 2.0]`，数值越大语速越快。默认值为 `1.0`。在短视频等快节奏场景下，略微加快语速能有效抓住观众注意力。推荐将此值设置为 `1.1` 或 `1.2`，以达到听感更佳、信息密度更高的效果。

* `drama_recap_config.pause_time`：控制配音的句间停顿时长，单位为毫秒。默认为 `120`。通过调整此参数，您可以改变解说的节奏，较大的值会使句子间停顿更明显，节奏更舒缓。


<span id="1eb1580a"></span>
#### 示例：语速加快至 1.1 倍，句间停顿 150 毫秒

```JSON
{
  "drama_script_task_id": "v03cd6g10067d8aotcchmjljad3qra7g",
  "recap_text": "...",
  "drama_recap_config": {
    "text_speed": 1.1,
    "pause_time": 150
  }
}
```


<span id="3b780e3c"></span>
### 自定义解说字幕样式

通过 `subtitle_config` 对象，您可以精细化控制新生成的解说字幕的样式，包括字体、大小、颜色、位置、描边等。

<span id="bb243c41"></span>
#### 示例：设置字幕为黄色、40px大小、带黑色描边

```JSON
{
  "drama_script_task_id": "v03cd6g10067d8aotcchmjljad3qra7g",
  "recap_text": "...",
  "subtitle_config": {
    "disable_subtitle": false,
    "font_size": 40,
    "font_color": "#FFFF00FF",
    "border_color": "#000000FF",
    "border_width": 2,
    "align_type": 1
  }
}
```


<span id="a570e925"></span>
### 擦除原视频字幕

如果您的原始视频包含内嵌的硬字幕，为了避免新生成的解说字幕与其重叠，造成画面杂乱，您可以使用字幕擦除功能。


* `erase_subtitle`：设置为 `true` 即可开启字幕擦除功能。

* `erase_mode`：指定擦除模式。目前仅支持 `standard`（标准版），在效果和效率之间取得了良好平衡。


<span id="4aaf4695"></span>
#### 示例：开启标准模式的原字幕擦除

```JSON
{
  "drama_script_task_id": "v03cd6g10067d8aotcchmjljad3qra7g",
  "recap_text": "...",
  "erase_subtitle": true,
  "erase_mode": "standard"
}
```


<span id="EgUaohOQSn"></span>
### 单任务批量生成多个解说视频

使用 `batch_count` 参数，您可以基于同一次剧本还原的结果，一次性生成多个版本的解说视频，适用于需要多样化素材的场景。

<span id="75628e13"></span>
#### 示例：一次性生成 3 个不同版本的解说视频

```JSON
{
  "drama_script_task_id": "v03cd6g10067d8aotcchmjljad3qra7g",
  "batch_count": 3,
  "drama_recap_config": {
    "auto_generate_recap": true,
    "style": "悬疑"
  }
}
```


任务完成后，`result.video_urls` 数组中将包含 3 个视频的下载地址。

### 使用限制


|限制项 |说明 |
|---|---|
|剧本还原的输入视频限制 |***视频来源**：支持公网 HTTP/HTTPS URL、本地上传 (`mediakit://`)、来源于火山引擎视频点播 (`vod://`) 和火山引擎对象存储 (`tos://`) 四种输入协议。关于各协议的具体说明和使用示例，请参阅[多源媒体输入与本地上传](https://www.volcengine.com/docs/6448/2536893)。<br><br>* **支持格式**：`mp4`, `flv`, `ts`, `avi`, `mov`, `wmv`, `mkv` 等主流视频格式，暂不支持 HLS (M3U8) 格式。<br><br>***数量与时长限制**：<br><br>* 单次任务支持传入 1 到 100 个视频文件。<br><br>   *单个视频时长不超过 120 分钟。<br><br>* 所有视频累计总时长不超过 90 分钟。<br><br>***分辨率限制**：输入视频的宽高必须保持一致。<br><br>* **内容要求**：<br><br>   *适用于**真人实拍的短剧、长剧、电影**等以人物对话和情节发展为核心的视频内容。不适用于**动画、纪录片、广告、直播录屏**等缺乏连贯真人剧情或人脸识别线索的视频。<br><br>* 视频必须包含清晰的**内嵌字幕（硬字幕）** 。 |
|解说词限制 |自定义解说词（`recap_text`）或 AI 生成解说词的期望长度（`text_length`）均不得超过 5000 字符。 |
